New Hope: "SPeeDY"

July 2012 - SPDY augments HTTP with several speed-related features that can dramatically reduce page load time

SPDY allows:

  • client and server to compress request and response header
  • multiple, simultaneously multiplexed requests over a single connection
  • the server to actively push resources to the client
  • SPDY allows client and server to compress request and response headers, which cuts down on bandwidth usage when the similar headers (e.g. cookies) are sent over and over for multiple requests.
  • SPDY allows multiple, simultaneously multiplexed requests over a single connection, saving on round trips between client and server, and preventing low-priority resources from blocking higher-priority requests.
  • SPDY allows the server to actively push resources to the client that it knows the client will need (e.g. JS and CSS)
